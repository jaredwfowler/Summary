What started as a simple openGL O-O style project turned into a 
mini-game engine. The main purpose of the project was to increase
my knowledge of openGL and provide a cool visual for the CSUN
CubeSat project.

Created using openGL 3.3 core, and visual studio c++. 

Simulator Reserved KEYS:
F1 -> Free Mouse
F2 -> Free View (Allows you to use the mouse and arrow keys to move around)
F3 -> Next Camera
F4 -> Hide/Show Text-2D
F5 -> Mute/UnMute Audio //Not implimented yet :-( TODO
F6 -> Max UPS/FPS
F7 -> On/Off Shadow Rendering
F11-> Show/Hide FPS
F12-> Show/Hide About
