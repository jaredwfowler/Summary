Stock market analytical tools which automates the "Darvas" and 
"Scharf" methods for hundreds of ticker symbols. Real-time stock
data is downloaded via cURL ( Page Scraping *** :p ) and analyzed. 
A CSV file is generated which can be used to determine which 
stocks or options to buy/sell.
